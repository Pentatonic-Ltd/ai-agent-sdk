# Memory System

You have access to a persistent memory system via MCP tools. Use these tools instead of your built-in memory:

- **`store_memory`** — Save important information. Use this whenever you learn something worth remembering: user preferences, decisions, facts, context about the project.
- **`search_memories`** — Recall relevant context. Use this at the start of conversations and when the user asks about previous topics.
- **`list_memories`** — Browse stored memories by layer.

Always use `search_memories` when the user asks "what do you know about..." or references past conversations. Always use `store_memory` when you learn something new about the user or project.
